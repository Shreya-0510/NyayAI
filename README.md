NyayAI is a **serverless, AI-driven platform** designed to simplify, analyze, and summarize complex legal documents efficiently.  
It demonstrates the integration of **cloud computing** and **modern AI models** to deliver secure, scalable, and real-time legal analytics.

---

## Highlights
- **Serverless Architecture:** Built using AWS Lambda, S3, API Gateway, and Cognito for **secure, scalable, and low-maintenance deployment**.  
- **AI-Powered Summaries:** Uses **Google Gemini 2.5 Flash model** to generate concise summaries and highlight critical clauses.  
- **Client-Side Text Extraction:** Implements **pdfjs-dist** for local PDF/text processing, bypassing AWS Textract limitations.  
- **Secure User Management:** Enables signup, login, and authenticated document handling via **AWS Cognito**.  
- **Interactive Frontend:** React-based UI for document upload, summary visualization, and user-friendly interaction.

---

## Tech Stack
- **Frontend:** React, HTML, CSS  
- **Backend / Serverless:** AWS Lambda, API Gateway, S3, Cognito  
- **AI & ML:** Google Gemini 2.5 Flash model  
- **Utilities:** pdfjs-dist for client-side text extraction  
- **Deployment / Prototype:** Serverless cloud architecture  

---

## Project Idea
With the exponential growth of digital legal documents, NyayAI **automates extraction, summarization, and analysis** to reduce manual effort and improve accessibility.  
It combines **cloud intelligence** and **AI models** to deliver real-time insights while ensuring **data privacy and cost-efficient scalability**.
